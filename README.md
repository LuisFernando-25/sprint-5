# Sprint 5 — Análise de Anúncios de Venda de Carros

Este projeto contém um aplicativo web desenvolvido com Streamlit para explorar
um conjunto de dados de anúncios de venda de carros nos EUA (`vehicles_us.csv`).

## Funcionalidades do aplicativo

- Exibe um cabeçalho com o título da análise.
- Botão para gerar um **histograma** da quilometragem (`odometer`) dos veículos anunciados.
- Botão para gerar um **gráfico de dispersão** relacionando preço (`price`) e quilometragem (`odometer`).

## Tecnologias utilizadas

- Python
- pandas
- plotly-express
- Streamlit

## Como executar localmente

1. Crie e ative um ambiente virtual.
2. Instale as dependências: `pip install -r requirements.txt`
3. Rode o app: `streamlit run app.py`
